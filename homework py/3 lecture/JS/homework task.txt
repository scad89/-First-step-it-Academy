Домашнее задание 1
Создать функцию которая высчитывает площадь прямоугольник, треугольника или круга в зависимости от входных параметров. 
Применить функцию для разных фигур(по 2 фигуры каждого типа)

Входные параметры:
 тип - 'triangle', 'square', 'circle'
объект с параметрами для конкретной фигуры:
прямоугольник - длины сторон {width, height } 
круг - радиус {r}
треугольник - длины сторон {a, b, c}


Домашнее задание 2
Создать функцию, которая отвечает на вопрос - является ли входной параметр палиндромом(примеры - "казак", "шалаш", 
"заказ"). При решении этой задачи нельзя пользоваться циклами.
Вывести результаты для 5 палиндромов и 5 не палиндромов.
Входной параметр - строка.
Выход - слово "Yes" или "No".

Получить результаты  для входного параметра: 5, 10, 20, 30
